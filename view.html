Views

Elegant uses Twig Templating Engine to power views.

All of your views will live in app/views directory. Just create any file in there.

Now if you want to render a view from the controller you can do: 

<?php
class Home
{
	public function index()
	{
		return View::render('home.html', [
			'Framework' => 'Elegant'
		]);
	}
}

And from the app/views/home.html you could just do: The framework name is {{ Framework }}!. And it will obviously render The framework name is Elegant!

To find out more about templating checkout the twig documentation.
