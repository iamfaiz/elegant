Controllers

A Controller is simply a class file that is named in a way that can be associated with a URI.

For example: example.com/home/index/hello will execute the index method of the Home class and give hello as the first argument. echo new Home()->index('hello');. Index is the default method it will execute if no other method is specified in the url. For example in the case of: example.com/home It will execute: echo new Home()->index(); You can also change the default controller in app/cofig/config.php. By default its Home. So if the url is example.com it will execute: echo new Home()->index();
All of the controllers live in app/controllers directory.
